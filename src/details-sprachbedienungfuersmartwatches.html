Sprachbedienung für Smartwatches

Der Ansatz in diesem Projekt war, dass über eine beiläufige Interaktion mit einer mobilen Anwendung ein großer Datensatz an Landmarken auf einem Campus-Gelände gesammelt werden kann. Fokussiert wurde hierbei ein Assistenzsystem für eine Smartwatch, bei der über eine Spracheingabe Metadaten der jeweiligen Landmarken erfasst werden. In wenigen Schritten können die Anwender diese Informationen mittels der Anwendung in eine Datenbank eintragen. Diese Landmarken wurden im Anschluss für eine sinnvolle Fußgängernavigation innerhalb des Campus-Geländes eingesetzt.

Im Vordergrund der Konzeptarbeit stand ein einfaches Dialogdesign, mit dessen Hilfe der Name, die Position sowie optionale Zusatzinformationen der Landmarke erhoben werden können. Mit einem Set an übergeordneten Sprachbefehlen kann die Eingabe korrigiert oder die Erhebung beendet werden. Die Ergebnisse dieser Arbeit wurden auf der International Conference on Mobile and Ubiquitous Multimedia vorgestellt und veröffentlicht.
